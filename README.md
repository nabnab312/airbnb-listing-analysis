# airbnb-listing-analysis
A data analysis project exploring Airbnb listings in New York City 2024, focusing on pricing, availability, room types, and neighborhood insights using Python, Pandas, and Matplotlib.

# Airbnb Listing Analysis (NYC 2024)

## Project Overview
This project focuses on Exploratory Data Analysis (EDA) and Data Visualization of Airbnb listings in New York City (2024). The analysis aims to uncover pricing trends, room type distribution, and neighborhood insights.

## Objectives
- Analyze Airbnb pricing patterns
- Explore room type distribution
- Identify popular neighborhoods
- Understand listing availability

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Insights & Conclusion

## Key Findings
- Manhattan has the highest average listing prices
- Entire home/apartment dominates the market
- Brooklyn and Manhattan have the largest listing volume

## Files
- `Main.ipynb` → Main notebook analysis
- `airbnb.csv` → Dataset used in the project
